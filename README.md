# Qt Image Loader

# Description

This is a Qt C++ application that allows users to load and display images with proper scaling.

# Features

* Load image from local system
* Display image in Qt UI
* Maintains aspect ratio
* Simple and user-friendly interface

# Technologies Used

* Qt Framework (C++)
* Qt Widgets

# How to Run

1. Open the project in Qt Creator
2. Build the project
3. Run the application
